# Security Policy

## Supported Versions
This repository is a reference techstack. Security fixes are applied to the latest `main` branch and to tagged releases.

## Reporting a Vulnerability
If you believe you have found a security vulnerability, do not open a public issue.

Please report privately by emailing the repository owner listed in the GitHub organization or by using GitHub's private security advisory flow (Security â†’ Advisories).

Include:
- a clear description of the issue,
- steps to reproduce,
- affected file paths,
- and the expected impact.

We aim to acknowledge reports within 72 hours.
